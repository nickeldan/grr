Determines if a string contains a substring which matches the regex. 

The function will stop processing characters when either it exhausts the specified length, a newline is encountered, or a non-\/printable character is encountered with tolerate\+Nonprintables set to false. If tolerate\+Nonprintables is true, then the function will skip over such characters and treat them as beginning and end of lines. For example, if the regex is \char`\"{}$^\wedge$a+\char`\"{} and the string is \char`\"{}aa\textbackslash{}x00aaad\char`\"{}, then both sequence of \char`\"{}a\char`\"{}\textquotesingle{}s will be considered matches. However, they will be considered separate matches and so $\ast$end-\/$\ast$start will be 3.
